# MESTO_BACKEND
### Backend for MESTO project created during Ya.Practicum course "WEB developer"
##### The goal of this project was to implement the server API for the project [MESTO_REACT_AUTH](https://github.com/PolinaLashchanka/react-mesto-auth).
##### The project implements methods for managing user accounts and photos.
#### Technological stack
- Node.js
- MongoDB
- Express.js
#### Installation:
```bash
git clone git@github.com:PolinaLashchanka/express-mesto-gha.git
```
